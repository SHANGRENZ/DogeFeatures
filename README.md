# DogeFeatures

一个Minecraft Spigot插件，包含一些小功能。

#### 功能

- 投票重启

- 投票封禁

- 新玩家进服消息提示

- op进服时自动隐藏

- 一些奇怪的恶趣味(可关闭)

#### 权限

> 重启指令 /restartvote
> 
> ```yml
> restartvote.use
> ```

> 对玩家创建封禁投票 /banvotecreate
> 
> ```yml
> banvote.create
> ```

> 

> 对玩家投票已创建的封禁 /banvote
> 
> ```yml
> banvote.vote
> ```
